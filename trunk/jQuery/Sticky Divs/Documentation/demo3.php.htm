    Meerkat Demo 3

    The following options have been set for this Meerkat function.

    meerkat({
  showMeerkatOnLoad: 'true',
  close: '.enter-site',
  animation: 'fade',
  animationSpeed: 1200,
  meerkatPosition: 'bottom',
  background: '#FFF url(images/top-bg.png) repeat-x 0 0',
  height: '100%'
});

    In this demonstration we're using Meerkat as an "entry" or "splash" page. The user would come to the URL and see a landing page with the Meerkat option "close" assigned to an anchor element with the class "enter-site".

    In this particular demonstration, I did not use the "dontShow" option. I would not recommend this, for it would really annoy the page viewer. Instead of assigning the enter-site class to the "close" option, I would assign it to the "dontShow" option. This way it doesn't pop up everytime the user reloads the page. You could simply set your "dontShowExpire" option to 1 (dontShowExpire: 1), for instance, and it wouldn't show the entry page for 24 hours. You could also leave the dontShowExpire default of 0 and the user would not see your entry page again until they restarted their browser and visited the URL again.

    Note: If you're not seeing the Meerkat demo, you've activated the "dontShow" option and need to remove the "dontShow" cookie.

Default Settings

  • showMeerkatOnLoad: 'false'
  • close: 'none'
  • dontShow: 'none'
  • dontShowExpire: 0
  • removeCookie: 'none'
  • meerkatPosition: 'bottom'
  • animation: 'slide'
  • animationSpeed: 'slow'
  • height: 'auto'
  • background: 'none'
