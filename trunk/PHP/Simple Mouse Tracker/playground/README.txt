Edit 'config.php' in order to change the administrator password. 

If you don't put the 'playground' folder in the same dir as 'logs_xml' and 'logs_html', 
you'll need to point to these logs folders correctly in 'config.php' (thery are 
relatives to the 'playground' folder).


You'll wish to customize 'head.html' with your site's information and 
point to your custom 'smt_playground.css' and/or 'favicon.ico' files.
