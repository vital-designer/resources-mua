    Meerkat Demo 2

    The following options have been set for this Meerkat function.

    meerkat({
  close: '.close-meerkat',
  animation: 'fade',
  animationSpeed: 750,
  meerkatPosition: 'bottom',
  background: '#71adb1',
  height: '110px'
});

    In this demonstration we're using Meerkat as a promotional tool. If the user clicks the "x" button, at the right of Meerkat, it will close it; but Meerkat will show again upon a page refresh or reload. If the user clicks the "Don't Show Again" link, based on the settings set, Meerkat will not show again until the user closes their browser and returns to this page.

    If we change the meerkatPosition from bottom to top:

    meerkat({
  close: '.close',
  dontShow: '.dont-show',
  animation: 'slide',
  animationSpeed: 500,
  dontShowExpire: 0,
  meerkatPosition: 'top',
  background: '#2e2a22 url(images/meerkat-bg.png) repeat-x 0 0',
  height: '100px'
});

    We get this instead.

    Note: If you're not seeing the Meerkat demo, you've activated the "dontShow" option and need to remove the "dontShow" cookie.

Default Settings

  • showMeerkatOnLoad: 'false'
  • close: 'none'
  • dontShow: 'none'
  • dontShowExpire: 0
  • removeCookie: 'none'
  • meerkatPosition: 'bottom'
  • animation: 'slide'
  • animationSpeed: 'slow'
  • height: 'auto'
  • background: 'none'
