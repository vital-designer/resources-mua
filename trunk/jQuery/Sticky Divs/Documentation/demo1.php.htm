    Meerkat Demo 1

    The following options have been set for this Meerkat function.

    meerkat({
  close: '.close',
  dontShow: '.dont-show',
  animation: 'slide',
  animationSpeed: 500,
  dontShowExpire: 0,
  meerkatPosition: 'bottom',
  background: '#2e2a22 url(images/meerkat-bg.png) repeat-x 0 0',
  height: '100px'
});

    In this demonstration we're using Meerkat as a home for Google Adsense. If the user clicks the "x" button, at the right of Meerkat, it will close it; but Meerkat will show again upon a page refresh or reload. In this particular demonstration, there isn't a "dontShow" option set, so the user does not have the ability to hide meerkat. I certainly wouldn't recommend this, for it will really annoy your page viewers, but maybe that's what you're going for? :)

    If we change the meerkatPosition from bottom to top:

    meerkat({
  close: '.close-meerkat',
  animation: 'fade',
  animationSpeed: 750,
  meerkatPosition: 'top',
  background: '#71adb1',
  height: '110px'
});

    We get this instead.

    Note: If you're not seeing the Meerkat demo, you've activated the "dontShow" option and need to remove the "dontShow" cookie.

Default Settings

  • showMeerkatOnLoad: 'false'
  • close: 'none'
  • dontShow: 'none'
  • dontShowExpire: 0
  • removeCookie: 'none'
  • meerkatPosition: 'bottom'
  • animation: 'slide'
  • animationSpeed: 'slow'
  • height: 'auto'
  • background: 'none'
